Embed Chew videos into Wordpress with one simple shortcode.

== Description ==

Chew is a Live and VOD platform for Artists, DJs and Labels. With this plugin you can embed Chew videos into Wordpress with one simple shortcode. [chew video="XXXXXXXXXX" height="500px" width="300px" autoplay="yes"]

== Installation ==

To install the plugin simply

1. Upload the plugin zip file or install from the 'Add New' plugins interface
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[chew video="XXXXXXXXXX" height="500px" width="300px" autoplay="yes"]` in any post replacing XXXXXXXXXX with your Chew.tv video id (found in the url of the video page)
